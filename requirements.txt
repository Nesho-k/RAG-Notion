# dans os.environ au démarrage du script. Évite d'écrire les clés API en dur.

# streamlit : framework pour créer des apps web en Python pur.

# openai : SDK officiel OpenAI — utilisé dans enrich_corpus.py pour générer les résumés.

# requests : effectue des requêtes HTTP pour télécharger les pages web.

# beautifulsoup4 : parse le HTML des pages pour en extraire le texte.
